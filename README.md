Pasos para Ejecutar
Clonar el repositorio:

Bash
git clone https://github.com/Chris103105/Investigacion_Aplicada_2_DPS_Catalogo.git


Instalar dependencias:

Bash
npm install


Iniciar el servidor de desarrollo:

Bash
npx expo start

Prueba en dispositivo  Escanear el código QR que aparecerá en tu terminal con la aplicación Expo Go.
